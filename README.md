# Employee Salary Analysis Using SQL

## Project Overview

This project analyzes employee salary data using SQL queries. The objective is to extract meaningful insights from employee records using filtering, aggregation, grouping, sorting, and conditional logic.

## Dataset

The dataset contains:

* Employee ID
* Employee Name
* Department
* Salary
* Experience

## SQL Concepts Used

* SELECT
* WHERE
* ORDER BY
* GROUP BY
* COUNT()
* AVG()
* CASE WHEN

## Business Questions Answered

1. Who earns more than ₹50,000?
2. What is the average salary by department?
3. Which employee has the highest salary?
4. How many employees work in each department?
5. How can employees be categorized based on salary?

## Key Insights

* IT department has the highest-paid employees.
* Salary distribution varies across departments.
* Employees can be segmented into High, Medium, and Low salary groups.
